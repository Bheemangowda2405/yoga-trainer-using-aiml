## Responsive Behavior

### Desktop (> 768px)

- Filter buttons in a row
- Summary cards in 3 columns
- Full date format
- Comfortable spacing

### Tablet (768px - 480px)

- Filter buttons in a row (smaller)
- Summary cards in 2 columns
- Abbreviated date format
- Reduced spacing

### Mobile (< 480px)

- Filter buttons stacked or scrollable
- Summary cards in 1 column
- Short date format
- Compact spacing
