# Yoga Control Bar

## Overview
The Yoga Control Bar is a minimalistic web application designed to enhance your yoga practice experience. It features a simple control interface with Play/Pause and Stop buttons, allowing users to easily manage their sessions.

## Features
- **Play/Pause Toggle**: Start or pause your yoga session with a single button.
- **Stop Button**: Instantly stop the session and reset the controls.
- **Modern Design**: Clean and minimalistic interface for a distraction-free experience.

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd yoga-control-bar
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage
1. Open `src/index.html` in your web browser.
2. Click the Play/Pause button to start or pause your yoga session.
3. Click the Stop button to end the session and reset the controls.

## Development
- The HTML structure is located in `src/index.html`.
- Styles are defined in `src/styles/control-bar.css`.
- JavaScript functionality can be found in `src/scripts/control-bar.js`.
